## Overview

This beginner-friendly article explains what goes on "behind the scenes" of artificial intelligence. It covers the basic definition of AI, the step-by-step process of how AI works, the four main types of AI, and the key disciplines that make up the field.

## What is Artificial Intelligence?

**Definition:** Artificial intelligence is the theory and discipline of programming computer systems to learn from and spot patterns in data sets.

### Core Characteristics
- Advanced algorithms and models that perform human tasks
- Learns from data rather than relying only on fixed rules
- Recognizes patterns to do tasks more efficiently and effectively

### What AI Enables
- Recognizing speech or images
- Making decisions
- Automating tasks
- Powering virtual assistants
- Generating transcripts
- Creating content (with generative AI)

### Real-World Examples
| Application | How AI Works |
|-------------|--------------|
| Netflix | Recommends TV shows and movies based on your preferences |
| Waymo | Powers self-driving cars |
| ChatGPT/Google Gemini | Generates content based on user prompts |
| Zoom | Creates automatic transcripts of calls |

## How AI Works: The 5-Step Process

To create AI, developers must: define the problem, determine outcomes, organize the data set, choose appropriate technology, and test solutions.

### Step 1: Input

**What happens:**
- Data is collected from various sources (text, audio, video, etc.)
- Data is sorted into categories
- Protocol and criteria are created for which data will be processed

**Key activities:**
- Gathering data from multiple sources
- Categorizing data (readable vs. non-readable by algorithms)
- Establishing processing rules

---

### Step 2: Processing

**What happens:**
- AI decides what to do with the collected data
- AI sorts and deciphers data using learned patterns
- System recognizes similar patterns in new data

**Key activities:**
- Pattern recognition
- Data sorting and interpretation
- Comparing new data to learned patterns

---

### Step 3: Outcomes

**What happens:**
- AI uses complex patterns to predict outcomes
- System determines if data is a "pass" or "fail" (matches previous patterns or not)
- Predictions are generated for decision-making

**Applications:**
- Customer behavior prediction
- Market trend analysis
- Decision support

---

### Step 4: Adjustments

**What happens:**
- When data is considered a "fail," AI learns from the mistake
- Process is repeated under different conditions
- Algorithm rules may be adjusted

**Key activities:**
- Learning from errors
- Refining algorithms
- Returning to outcomes step if needed
- Aligning with current data conditions

---

### Step 5: Assessments

**What happens:**
- AI synthesizes insights from the data set
- Predictions are made based on outcomes and adjustments
- Feedback is incorporated into the algorithm

**Key activities:**
- Synthesizing insights
- Making predictions
- Incorporating feedback
- Preparing for future tasks

## How Does Generative AI Work?

**Powered by:** Large Language Models (LLMs)

**Key characteristics:**
- Complex machine-learning models created from algorithms
- Trained on massive data sets with deep learning
- Produces or generates **new content** (not just predicts patterns)

**Examples:** ChatGPT, Microsoft Copilot

**Important note:** While the applications are new, many core concepts have existed for much longer.

## The Four Main Types of AI

AI can be classified by its capabilities as the technology evolves:

### 1. Reactive Machines
| Aspect | Description |
|--------|-------------|
| **Memory** | No memory |
| **Focus** | Task-specific |
| **Behavior** | An input always delivers the same output |
| **Current status** | Exists today |

---

### 2. Limited Memory Machines
| Aspect | Description |
|--------|-------------|
| **Memory** | Can store and use past data |
| **Focus** | Learning and improving over time |
| **Behavior** | Imitates how brain neurons work; gets smarter with more data |
| **Current status** | Exists today (most current AI) |

---

### 3. Theory of Mind
| Aspect | Description |
|--------|-------------|
| **Memory** | Advanced |
| **Focus** | Understanding others' thoughts and emotions |
| **Behavior** | Would behave differently in relation to those around it |
| **Current status** | Does not yet exist |

---

### 4. Self-Awareness
| Aspect | Description |
|--------|-------------|
| **Memory** | Full consciousness |
| **Focus** | Understanding own existence |
| **Behavior** | Realizes its state of being; predicts others' feelings |
| **Current status** | Does not yet exist |

## Disciplines That Make Up AI

Understanding the relationship between AI and its subfields:

### Machine Learning
- **What it is:** A subset of AI incorporating computer science, mathematics, and coding
- **Focus:** Developing algorithms that help machines learn from data
- **Goal:** Predict trends without human assistance

---

### Deep Learning
- **What it is:** A discipline of AI that imitates the human brain
- **Focus:** Learning from how the brain structures and processes information
- **Key feature:** Can learn from unstructured data without supervision

---

### Neural Networks
- **What it is:** A deep learning technique designed to resemble the human brain's structure
- **Focus:** Performing calculations and creating outputs
- **Application:** Processing large data sets

---

### Natural Language Processing (NLP)
- **What it is:** AI that enables computers to understand human language
- **Focus:** Spoken and written language comprehension
- **Applications:** Text recognition, speech recognition on devices

---

### Computer Vision
- **What it is:** An interdisciplinary field focused on visual understanding
- **Focus:** How computers gain understanding from images and videos
- **Application:** Automating activities the human visual system performs

### Connecting to Your Seminar Frameworks

| Coursera Concept | Framework Connection |
|------------------|---------------------|
| Processing & Patterns | Why clear prompts (Description/TCREI) matter |
| Adjustments & Learning | Supports iterative collaboration (Augmentation modality) |
| Outcomes & Predictions | Why Discernment/human review is essential |

### Simple Explanation for Students

> "AI works by: (1) taking in data, (2) finding patterns in that data, (3) making predictions based on those patterns, (4) learning from mistakes, and (5) getting better over time. When you write a prompt, you're providing the input. The AI processes it based on patterns it learned from training. Your job is to evaluate whether the output is good enough—and if not, adjust your prompt and try again."
